## Building Locally

The project is designed to be compiled on [Overleaf](https://www.overleaf.com). To build locally, you need a LaTeX distribution (e.g. [TeX Live](https://www.tug.org/texlive/) or [MiKTeX](https://miktex.org/)) and run:

```bash
pdflatex main.tex
```
